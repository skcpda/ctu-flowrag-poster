The name of this welfare scheme is Hamaal Hetu Joota, Huk Evan Mahila Hamaal Hetu Soopa Aur Tokari Sahaayata Yojana. It is also known as HHJMHHST. Labor Department of Chhattisgarh State started this scheme &amp;quot;Hamaal Hetu Joota, Huk Evan Mahila Hamaal Hetu Soopa Aur Tokari Sahaayata Yojana&amp;quot; for unorganized worker on  11 March 2015. The main objective of the scheme to empower unorganized worker by providing financial assistance The scheme targets beneficiaries such as Individual. It is specifically implemented in Chhattisgarh. Additionally, the nodal department is Labour Department.

The scheme provides the following benefits: For shoe, hook for lifting sack for Hamal and soup and basket for female Hamal :  â‚¹ 500/per beneficiary.

Eligibility criteria include: Applicant should be registered in any district of the state.
If applicant received from any other parallel scheme of the State Government, then applicant is not eligible for this scheme.
Beneficiaries aged between 18 years to 60 years will be eligible under the scheme. The application process is as follows: Unregistered Applicant :

All the applicant have to visit the official website Chhattisgarh Labour Department. On the home page click on &quot;Apply&quot; under &quot;Chhattisgarh Unorganized Workers State Social Security Board&quot;.Enter the following details

   Select Group Name : &quot;Asangathit Marmakaar Mandal&quot;
   Select Service : &quot;Asangathit Shramik Panjikaran&quot;
   What do you want to do :  &quot;Aavedan&quot;
   Click next.

Check the eligibility criteria for registration in Chhattisgarh Unorganized Workers State Social Security Board and click next. 
Fill out the online &quot;Registration form&quot;.
Submit.



Already Registered Applicant :

All the applicant have to visit the official website Chhattisgarh Labour Department. On the home page click on &quot;Apply&quot; under &quot;Chhattisgarh Unorganized Workers State Social Security Board&quot;.Enter the following details

   Select Group Name : &quot;Asangathit Marmakaar Mandal&quot;
   Select Service : &quot;Yojana&quot;
   What do you want to do :  &quot;Aavedan&quot;
   Click next.

Select your district name and provide old/new registration number and click next.
Select the scheme name. 
Fill out the online with required details. 
Submit.
